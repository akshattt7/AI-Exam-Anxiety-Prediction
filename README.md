# AI-Exam-Anxiety-Prediction
# 🧠 AI-Based Exam Anxiety Prediction System

## 📌 Project Overview
This project is an AI-based system that predicts the level of exam anxiety in students based on various input factors. It helps in identifying students who may be under stress so that proper steps can be taken.

---

## 🎯 Objective
- To analyze student data  
- To predict exam anxiety level using Machine Learning  
- To help in early detection of stress  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit (for web app)

---

## 📊 Features
- User-friendly interface  
- Takes student input data  
- Predicts anxiety level instantly  
- Displays result in simple format
